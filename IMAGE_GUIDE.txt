부부피오피 랜딩페이지 이미지 가이드 (업로드용)

목표
- “빠르게 제작 가능한 업체” 인상을 강하게 전달
- 모바일에서 ‘문의’ 버튼을 누르기 쉽게, 이미지가 텍스트를 방해하지 않게

공통 규칙
- 파일 위치: public/images/
- 코드에서 사용하는 경로: /images/파일명.jpg (또는 .png/.webp)
- 권장 포맷: webp > jpg > png(로고/투명 필요할 때만)
- 권장 용량(모바일): 1장당 100KB~400KB 정도(가능하면 더 작게)
- 권장 해상도: 최소 1200px(긴 변 기준) 이상 (확대/리사이즈에 유리)
- 중요한 글자(가격/문구)는 이미지에 넣지 말고 텍스트로 표현하는 편이 안전

1) 메인 히어로 “대표 이미지 영역”
- 위치: 메인 첫 화면(큰 문구 아래, CTA 버튼 위)
- 화면 비율: 4:3 (현재 코드 aspect-[4/3])
- 권장 픽셀: 1200x900 또는 1600x1200
- 파일명 추천: hero.jpg 또는 hero.webp
- 추천 이미지:
  - 작업 현장/장비(프린터, 커팅기, 작업대) + 결과물 일부가 같이 보이는 사진
  - “당일 제작/빠른 제작” 느낌이 나도록 손/작업 중인 컷이면 더 좋음
  - 너무 어둡거나 복잡한 배경은 피하기(텍스트/CTA 집중)

2) 메인 “작업 스냅 & 블로그” 카드 썸네일
- 위치: 블로그 최신글 3개 카드 상단 썸네일
- 화면 비율: 4:3 (현재 코드 aspect-[4/3])
- 실제 이미지 소스: 네이버 RSS의 글 본문 첫 이미지(img src)를 자동 추출해 표시
- 내가 준비할 이미지: 별도 준비 필요 없음 (블로그 글에 첫 이미지를 잘 넣으면 자동으로 썸네일로 잡힘)
- 팁:
  - 블로그 글의 첫 이미지를 ‘결과물 사진’으로 올리면 랜딩에도 같은 이미지가 썸네일로 뜸
  - 첫 이미지에 글자가 너무 작게 들어가면 모바일에서 안 보일 수 있음(제품/결과물 위주 추천)

3) 각 서비스 페이지(스티커/단체티/DTF/배너/명함/굿즈/선거용품/기타) 상단 “대표 이미지”
- 위치: 각 페이지 상단(PageHero)의 대표 이미지 영역
- 화면 비율: 4:3 (현재 코드 aspect-[4/3])
- 권장 픽셀: 1200x900
- 파일명 추천(예시):
  - sticker-hero.jpg
  - tshirt-hero.jpg
  - dtf-hero.jpg
  - banner-hero.jpg
  - card-hero.jpg
  - goods-hero.jpg
  - election-hero.jpg
  - another-hero.jpg
- 추천 이미지:
  - 해당 품목의 “완성 결과물 1장”을 크게, 배경은 단순하게
  - 가능한 한 정면/45도 각도, 제품이 화면의 60~80% 차지하도록

4) 각 서비스 페이지 “제작 사례” 슬라이드(3개씩 보이고 넘김)
- 위치: 각 페이지 하단 제작 사례 섹션(ServicePortfolioSection)
- 화면 비율: 4:3 (현재 코드 aspect-[4/3])
- 권장 픽셀: 1200x900
- 기본 노출: 한 페이지에 3개 카드가 보이고, 버튼/점으로 페이지 전환
- 파일명 추천(예시):
  - sticker-1.jpg, sticker-2.jpg, sticker-3.jpg
  - tshirt-1.jpg, tshirt-2.jpg, tshirt-3.jpg
  - dtf-1.jpg, dtf-2.jpg, dtf-3.jpg
  - banner-1.jpg, banner-2.jpg, banner-3.jpg
  - card-1.jpg, card-2.jpg, card-3.jpg
  - goods-1.jpg, goods-2.jpg, goods-3.jpg
  - election-1.jpg, election-2.jpg, election-3.jpg
  - another-1.jpg, another-2.jpg, another-3.jpg
- 추천 이미지(제작 사례 3장 구성 예시):
  - 1) 대표 결과물(근접)
  - 2) 사용 장면(부착/착용/설치)
  - 3) 디테일 컷(코팅/재질/인쇄 품질)

5) 오시는 길(/location) “지도 영역”
- 위치: 오시는 길 페이지의 지도 placeholder
- 화면 비율: 16:9 (현재 코드 aspect-[16/9])
- 권장 픽셀: 1600x900
- 파일명 추천: map.jpg 또는 location-map.jpg
- 추천 이미지:
  - 네이버지도/카카오지도에서 캡처한 약도 이미지(주변 랜드마크 포함)
  - 주소/층수/주차 안내는 이미지에 너무 작게 넣지 말고, 페이지 텍스트로 같이 적는 걸 추천

촬영/준비 체크리스트 (사진 찍어서 보내주실 때)
- 히어로 1장(작업현장 느낌) + 서비스별 대표 1장 + 제작사례 각 3장(가능하면)
- 밝은 조명, 흔들림 없는 컷, 제품이 크게 보이게
- 개인정보(전화번호, 차량번호 등) 노출되면 가려서 촬영

보내주실 때 권장
- 파일명을 위 추천대로 맞춰서 보내기 (제가 바로 연결하기 쉬움)
- 예: hero.jpg / sticker-1.jpg / banner-hero.jpg 처럼
