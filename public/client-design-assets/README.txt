결제 고객 전용 시안 이미지 (수동 배포용)
============================================

이 폴더에 이미지를 넣은 뒤 Git에 커밋하고 다시 배포하면,
/client-designs 페이지(비밀번호 입장 후)에 표시됩니다.
웹에서 업로드하는 기능은 사용하지 않습니다.

파일 이름 규칙
--------------
  슬롯이름.확장자

슬롯 이름 (대문자, 예시):
  POSTER_A, POSTER_B, … POSTER_F   → 포스터
  PIKET_A  …                        → 피켓
  SUPPLIES_A …                      → 용품

허용 확장자: .webp, .jpg, .jpeg, .png, .gif
같은 슬롯에 여러 확장자를 넣지 마세요. 우선순위는 webp → jpg → jpeg → png → gif 입니다.

예시:
  POSTER_A.webp
  PIKET_C.jpg
  SUPPLIES_B.png
